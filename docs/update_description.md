### **Обновление бота / Update @ytttins_dl 1.1**

🇷🇺 **Что нового:**
- Добавлена новая команда для поиска песен на английском: `search <название>` (также работает привычное `найти <название>`).
- Теперь у бота есть удобные кнопки внизу экрана для быстрого доступа к поиску. Нажмите на них, чтобы получить подсказку!

🇬🇧 **What's new:**
- Added a new song search command: `search <song name>` (alongside the original `найти <song name>`).
- Now the bot has convenient buttons at the bottom for quick access to search. Tap them to get started!
